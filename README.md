# Roadmap: Machine Learning Engineer (Fase 1)

Este repositório tem como objetivo documentar meus estudos, anotações e projetos práticos durante a **Fase 1** de uma trilha focada em vagas de Machine Learning.

## Objetivo
Registrar minha evolução técnica ao longo dos próximos meses, consolidando conhecimentos práticos e teóricos exigidos pelo mercado. O roadmap completo que estou seguindo pode ser visualizado no arquivo `roadmap_tracker_ifood.html` presente neste diretório.

## Empresas Foco
Esta trilha foi estruturada com base nos requisitos e tecnologias comuns às vagas de Machine Learning Engineer em grandes empresas de tecnologia na América Latina, com foco especial em:
* **iFood**
* **Nubank**
* **Mercado Livre**
